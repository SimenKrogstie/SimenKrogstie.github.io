## About
The site presents a short professional profile, work experience, education, projects, and skills.  
It is intended as a lightweight, always-up-to-date alternative to a traditional PDF CV.

## Tech stack
- HTML
- CSS
- GitHub Pages

## Updating the site
All content is maintained directly in the HTML/CSS files.
Changes pushed to the `main` branch are automatically deployed via GitHub Pages.
